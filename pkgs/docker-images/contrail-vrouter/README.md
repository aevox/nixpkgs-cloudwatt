The vrouter agent container launch a QEMU vm and use a VDE network for the dataplane.

The following figure describes 2 virtual router containers.
```

 |    | +------+     |kernel module|   | agent |      |      |
 |    |              +-------------+   +-------+      |      |
 |    |                    |               |          |      |
 |    | VM                eth1            eth0        |      |
 |    +--------------------|---------------|----------+      |
 |                        VDE            SLIRP               |
 |                         |               |                 |
 | container            volume            eth0               |
 +-------------------------|---------------|-----------------+
```
